简单说下使用，打开控制台后会加载一系列功能
主要功能为三个模块：生命值监测、Attack、LevelUP
生命值监测模块保证了生命值不低于DefHP定义的值(默认为1000)，该模块默认开启
Attack 模块，自动修正连击率和必杀率(默认待机，需要Switch Attack开启，且在极少情况下会引起闪退)
LevelUP模块 自动补正基础经验值(40000000),快速升级(默认待机，需要Switch LevelUP开启)


其次，提供了Set <ValueName> <Value>命令来修改包括但不限于团队生命值、食卷、基础经验等游戏内的值和一些控制台内部参数

（2）

打开后生命值监测模块默认生效（生命值低于1000自动补正），主要功能命令有两条，分别是switch <模块名>，用来切换模块状态，还有Set <数值名> <要修改为的值>，来修改游戏内的值，具体数值、模块名参见help命令帮助页
